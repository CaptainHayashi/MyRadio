public static boolean
# requirePermissionAuto( String $service, String $module, String $action, boolean $require = true )

Checks if the user has the given permissions required for the given Service/Module/Action combination

Checks if the user has the given permissions required for the given Service/Module/Action combination

The query needs a little bit of explaining.
The first three WHERE clauses just set up foreign key references - we're searching by name, not ID.
The next three WHERE clauses return exact or wildcard matches for this Service/Module/Action combination.
The final two AND NOT phrases make sure it ignores wildcards that allow any access.

Parameters

$service
String
$service The Service to check permissions for
$module
String
$module The Module to check permissions for
$action
String
$action The Action to check permissions for
$require
boolean
$require If true, will die if the user does not have permission. If false, will just return false

Returns

boolean
True on required or authorised, false on unauthorised
