Class APCProvider

APCProvider provides in-memory caching for PHP resources to increase page load times

APCProvider was the first CacheProvider implementation in MyURY. It enables Models to send cache commands to it which are then stored using the APC plugin automatically. In order for this class to work correctly, the server needs the APC PHP plugin installed on the server. It will throw an Exception if it is not.
