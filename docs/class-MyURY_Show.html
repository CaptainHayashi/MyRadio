Parameters

$params
Array
$params An array of Show properties compatible with the Models/Scheduler/showfrm Form: title: The name of the show
description: The description of the show
genres: An array of 0 or more genre ids this Show is a member of
tags: A string of 0 or more space-seperated tags this Show relates to
credits: An array of 1 or more memberids of people related to the Show
credittypes: An array of identical size to credits, identifying the type of relation to the Show
showtypeid: The ID of the type of show (see schedule.show_type). Defaults to "Show" location: The ID of the location the show will be in title, description, credits and credittypes are required fields. As this is the initial creation, all tags are approved by the submitted so the show has some initial values
